# Habit Tracker

Full-stack habit tracker. Next.js + tRPC.

## Highlights

- Custom GitHub-style heatmap to track consistency across habit.
- Optimistic mutations: instant completion toggles, in-flight query cancellation, undo without state corruption.
- Streak: cross-month streaks, monthly aggregation, timezone-safe date math.
- tRPC backend: transaction-safe mutations, accurate "most recent completion" queries.

/notes is public. Architecture decisions, bugs, tradeoffs — all documented.

Built in the open.
